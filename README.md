# Windows-11-Product-Key-Selector
This script allows you to select and activate a Windows 11 product key.

# Features
Select from a list of Windows 11 editions and their corresponding product keys
Automatically activates the selected product key using slmgr /ipk and slmgr /ato
Checks if the script is being run as administrator before execution
# Requirements
Windows 11 operating system
Administrator privileges
# Usage
Download or clone the repository
Run the windows11_product_key_selector.bat script as administrator
Follow the prompts to select your desired Windows 11 edition
The script will automatically activate the selected product key
# Notes
Make sure to right-click on the script and select "Run as administrator" before executing
This script is intended for educational purposes only and should be used responsibly
Always verify that you have the appropriate license and rights to use the selected Windows 11 edition
